Notes

• The page finds identifiers other than x and y and treats them as parameters.
• For each parameter (e.g. k) the script defines k = seq(i, i, pmin, pmax, pstep) so Desmos draws many curves.
• If no parameter is found, the equation is plotted normally.
• If Desmos is blocked by an extension, the page falls back to an iframe of Desmos.
